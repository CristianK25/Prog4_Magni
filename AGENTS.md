# Reglas del Proyecto: Registro de Eventos

## Stack Tecnológico
- Vite
- TypeScript estricto
- Tailwind CSS
- React

## Reglas de Código
1. **TypeScript:** Prohibido usar `any`. Tipar siempre las props y retornos de funciones.
2. **Estilos:** Usa exclusivamente clases de Tailwind. No crees archivos `.css` adicionales a menos que sea estrictamente necesario para animaciones complejas.
3. **Componentes:** Usa componentes funcionales. Mantén la lógica separada de la vista cuando sea posible.
4. **Idioma:** Escribe el código, las variables y los comentarios en español (ya que es un TP de la facultad).
